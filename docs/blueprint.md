# **App Name**: Farmingo: Grow Together

## Core Features:

- Personalized Dashboard: Display a personalized dashboard with price snapshots, alerts, tasks, and a community feed preview to provide farmers with an at-a-glance overview of relevant information.
- Community Feed: Enable farmers to connect and engage with each other by posting updates, participating in threaded comments, and building a trusted network.
- Marketplace: Provide a platform for verified users to easily list and sell their products, browse available crops, and filter by location and other criteria.
- Crop Price Prediction: Offer a crop price prediction tool that uses machine learning to suggest optimal selling prices, incorporating factors like region, crop, variety, and historical data.  The tool explains which factors affect its output, enabling users to understand how each aspect influences price suggestion
- Crop Disease Diagnosis: Allow farmers to upload photos of crops to receive an AI-powered diagnosis of potential diseases, along with recommended steps and links to relevant community discussions.
- Direct Messaging: Enable direct communication between farmers, buyers, and experts with image attachments and negotiation history.
- User Verification: Implement a verification flow to ensure trustworthy sellers and community members, enhancing platform credibility. Requires the use of Firestore.

## Style Guidelines:

- Primary color: Deep Green (#2B6E4A) to convey trust and agriculture.
- Accent color: Warm Amber (#F59E0B) for call-to-actions and highlights.
- Background color: Soft Off-White (#F6F9F6) for a clean, modern feel.
- Headline font: 'Inter' (sans-serif) for a bold, legible look.
- Body font: 'Noto Sans' (sans-serif) for excellent multi-language support.
- Use a consistent line + filled icon set, such as lucide-react icons.
- Subtle elevation and scale animations for CTAs on hover/tap to provide positive feedback.
